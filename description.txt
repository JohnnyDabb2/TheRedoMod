This mod currently adds 9 items and 1 boss.  There is far more content coming, so downloading it now wouldn't be a bad idea.  If you have any questions, please join the discord server by clicking the homepage button!



CURRENT FEATURES

1) The Money Trough can be crafted with a piggy bank and 70 cloud, as long as you're standing near an Ancient Manipulator!

2) Princess Pinky is a new boss that can be spawned with a Pinky Crown.  If you want to know more information about this boss, let me know on the discord server.

3) You'll have a new starting item in your inventory, the Goodie Bag.  Right click on this item for a pleasant surprise!  Yes, this is based on the Starter Bag from Calamity.

4) One Invisible Block can be crafted with five Dirt Blocks at the Ancient Manipulator.  Two Invisible Platforms can be crafted with one Invisible Block.  These tiles are nearly impossible to find, so only use them if you know you want to!
